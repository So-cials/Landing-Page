# Socials Landing Page

Wiira Labs product marketing and beta signup landing page.

## Ownership

This project is a product of **Wiira Labs**.

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS

## Local development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
npm run preview
```

## Deployment (Netlify)

This repository includes Netlify configuration at the repo root (`netlify.toml`) for monorepo-style builds.

- Build command: `npm --prefix Landing-Page run build`
- Publish directory: `Landing-Page/dist`

## Forms

Netlify Forms are configured for the beta signup form named `beta-signup`.
